# cyberbullying_detection_tfidf
CSC - 502 Massive Datasets Project - Cyberbullying Detection Using TF-IDF Feature Extraction

Replication of ICITSI 2022 paper.
Dataset: Kaggle Cyberbullying Classification (~47k tweets)
Models: SVM and k-NN with Unigram, Bigram, Uni+Bigram TF-IDF
Language: Python | scikit-learn | NLTK

# How to Run
1. Install dependencies: pip install pandas numpy scikit-learn matplotlib seaborn nltk
2. Place cyberbullying_tweets.csv in the same folder as the notebook
3. Run all cells in order
